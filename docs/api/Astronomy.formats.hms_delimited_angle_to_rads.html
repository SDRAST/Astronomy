hms_delimited_angle_to_rads¶

Astronomy.formats.hms_delimited_angle_to_rads(angle)¶

As in VLA Calibrator List::
>>> hms_delimited_angle_to_rads('00h01m08.621563s')
0.0049903008842279899

:param angle : R.A. formatted like 00h01m08.621563s :type angle : str

float
