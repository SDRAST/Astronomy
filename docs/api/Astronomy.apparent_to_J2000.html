apparent_to_J2000¶

Astronomy.apparent_to_J2000(MJD, UT, ra, dec, longitude, latitude)¶

observed celestial coordinates to J2000

This converts observed RA to CIRS RA, and then transforms the coordinates to the FK5 frame.
