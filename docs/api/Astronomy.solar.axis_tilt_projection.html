axis_tilt_projection¶

Astronomy.solar.axis_tilt_projection(longitude, tilt)¶

Project of Z-axis of a tilted frame

The tilt is in the YZ plane towards -Y

:param longitude : longitude in XY plane from the X-axis (Y = 0) :type longitude : float

:param tilt : tilt angle measured CCW about X-axis :type tilt : flat

Returns:radians
