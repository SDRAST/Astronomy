parse_hms_delimited_angle¶

Astronomy.formats.parse_hms_delimited_angle(angle)¶

Parses RA strings delimited with h, m and s.

As in VLA Calibrator List: 00h01m08.621563s

Example:

>>> parse_hms_delimited_angle('00h01m08.621563s')
['00', '01', '08.621563']

:param angle : R.A. formatted like 00h01m08.621563s :type angle : str

Returns

list of str
