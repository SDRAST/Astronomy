SerializableBody¶

class Astronomy.Ephem.SerializableBody(info=None, name='', observer_info=None)¶

Bases: ephem.FixedBody

Simple extension to ephem.FixedBody that allows for conversion to and from a Python dictionary.

Examples:

Usage is similar to ephem.FixedBody:

body = SerializableBody(
    name="0521-365",
    info={
        "flux": {
            "C": "6.76",
            "K": "3.50",
            "L": "16",
            "S": "12.50",
            "W": "3.55",
            "X": "5.19"
        }
})
body._ra = 1.4092068106703093
body._dec = -0.6363220818328413
body.compute()
