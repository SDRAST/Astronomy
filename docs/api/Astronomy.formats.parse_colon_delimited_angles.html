parse_colon_delimited_angles¶

Astronomy.formats.parse_colon_delimited_angles(*args)¶

Parses angle strings delimited with colons

Input data like (‘1:38:48.0’,’41:24:23’) and it will return [1.64666666667 41.4063888889]

:param args : list of hexadecimal strings :type args : list of str

Returns

list of float
