source_start_stop¶

Astronomy.source_start_stop(HAstart, HAend, HAlimit)¶

Find when sources are up

This only handles tracks less than 24 hr long.

The result codes are:

A  - always up
N  - never up
R  - rises
RS - rises and sets
S  - sets
SR - sets and rises (up twive during a track

:param HAstart - HA at beginning of track :type HAstart - float or int

:param HAend - HA at end of track :type HAend - float or int

Returns:str
