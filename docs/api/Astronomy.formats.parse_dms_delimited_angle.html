parse_dms_delimited_angle¶

Astronomy.formats.parse_dms_delimited_angle(angle)¶

Parses decl. strings delimited with d, ‘ and “.

As in VLA Calibrator List: 19d14‘33.801860” Example:

>>> parse_dms_delimited_angle('''19d14'33.801860"''')
['19', '14', '33.801860']

:param angle : decl. formatted like 00h01m08.621563s :type angle : str

Returns:list of str
