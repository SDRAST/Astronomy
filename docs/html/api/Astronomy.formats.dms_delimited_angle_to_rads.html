dms_delimited_angle_to_rads¶

Astronomy.formats.dms_delimited_angle_to_rads(angle)¶

converts decl. formatted as 19d14’33.801860 to radians As in VLA Calibrator List: 19d14’33.801860” Example:

>>> dms_delimited_angle_to_rads('''19d14'33.801860"''')
0.33584886884199222

:param angle : decl. formatted like 00h01m08.621563s :type angle : str

Returns

float
