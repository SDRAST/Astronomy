great_circle_distance¶

Astronomy.great_circle_distance(central_angle)¶

Great circle distance in km on Earth from great circle angle

Notes

Assumes a spherical earth of radius 6372.795 km. This is accurate to about 0.5%

:param central_angle : angle subtended at the center of the earth :type central_angle : float

Returns

float
